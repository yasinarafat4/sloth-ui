## Sections

1. **Hero Section** - Eye-catching introduction with CTA buttons
2. **Social Proof** - Trusted by major companies (Delta, Boeing, SpaceX, etc.)
3. **Features** - 6 key features with icons and descriptions
4. **Statistics** - Animated counters showing key metrics
5. **Client Showcase** - Interactive tabs with image previews
6. **Testimonials** - Masonry grid layout with star ratings
7. **Contact** - Office locations across 4 countries
8. **FAQ** - Accordion-style frequently asked questions
9. **CTA** - Final call-to-action with app mockups
10. **Footer** - Site map, links, and scroll-to-top button

## Key Functionality

### Mobile Menu

- Hamburger icon toggles navigation
- Click outside to close
- Smooth animations

### Statistics Counter

- Intersection Observer triggers animations
- Counts up when scrolled into view
- Easing effect for smooth animation

### Client Tabs

- Click to switch between previews
- Fade transition effect
- Active state highlighting

### Testimonials Grid

- CSS column layout for masonry effect
- Dynamic star ratings (full, half, empty)

### FAQ Accordion

- Smooth scroll behavior
- Hover effects
